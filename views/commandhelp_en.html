1. Command trigger

Although the usual trigger is a user entering a command in chat, there are multiple ways to trigger a command.

1.1. Classic user command

The classic user command is a simple phrase that is used in the beginning of a chat message by a user. Other than common bots TTVST allows you to use spaces in this phrase, but it is highliy discuraged since this might end up in confusion in the response creation. There's no special way to create this trigger, simply enter your phrase into the command field. It is recommended that you use a symbol (e.g. !, ~) at the start of your command to have it not trigger accidentaly. Slahes (/) cannot be used as they are recognized by twitch as a command and will not be posted as a message. For this reason slashes are used by TTVST to mark other triggers (see further below).
